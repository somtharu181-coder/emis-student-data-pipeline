# EMIS Student Data Pipeline
This project automates the process of extracting student data from the EMIS portal. It logs into the system, takes user input (grade and year), scrapes the relevant data, cleans it, and saves it into a structured CSV file.

## 🚀 Features
- Automated login to EMIS dashboard
- Dynamic input (grade and year selection)
- Data scraping from secured portal
- Data cleaning and formatting
- Export to CSV file

## 🛠️ Tech Stack
- Python
- Requests/Selenium
- csv

## 📂 Workflow
1. Login to EMIS portal
2. Enter grade and year
3. Scrape student data
4. Clean and structure data
5. Save as CSV
